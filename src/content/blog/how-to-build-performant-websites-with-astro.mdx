This is a sample blog post about building performant websites with Astro.

Astro is a modern static site builder that delivers lightning-fast performance with a unique "zero JavaScript by default" architecture.

### Key Features

- **Component Islands:** Astro renders UI components in isolation, enabling partial hydration.
- **Framework Agnostic:** Use your favorite UI frameworks like React, Svelte, or Vue.
- **Content Collections:** Manage your content with type-safe Markdown and MDX.
